How to Use

<!-- 1. Include jQuery and FormRepeater --> <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script> <script src="https://cdn.jsdelivr.net/gh/yourusername/quick-component@main/public/formix.js"></script> <!-- 2. Create your template (hidden) --> <div id="my-template" class="form-repeat-box-item" style="display: none;"> <input type="text" class="form-repeat-field-ipt" data-field-name="name"> <button class="form-repeat-delete-btn">Remove</button> </div> <!-- 3. Create container --> <div id="my-list"></div> <!-- 4. Create add button --> <button class="form-repeat-add-btn" data-group-name="items" data-box-default="#my-template" data-box-list="#my-list"> Add Item </button> <!-- 5. That's it! FormRepeater auto-initializes -->
